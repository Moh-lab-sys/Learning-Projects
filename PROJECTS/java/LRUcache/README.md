# LRU Cache Implementation in Java

## Description
A fixed-size cache that removes least recently used items when full. Uses HashMap + Singly Linked List for O(1) get and put operations.

## Features
- put(key, value)
- get(key)
- Automatic eviction

## Data Structures
- HashMap<Key, Node>
- Singly Linked List

## How to Run
1. Open in IDE
2. Run Main.java

## Sample Output
get(1) -> 10
Cache: 1,2
